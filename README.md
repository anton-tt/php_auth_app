# PHP Auth App

Сервис регистрации и авторизации пользователей на PHP (+ СУБД PostgreSQL).

## Описание

Реализована следующая функциональность:

- Регистрация пользователей с уникальными полями:
  - Имя
  - Телефон
  - Email
  - Пароль (с подтверждением)
- Авторизация по email или телефону + паролю
- Проверка Google reCAPTCHA при входе
- Страница профиля, доступная только авторизованным пользователям:
  - Изменение личных данных (Имя, Email, Телефон)
  - Изменение пароля
- Выход из аккаунта с уничтожением сессии

## Технологии

- Нативный PHP (без фреймворков)
- PostgreSQL
- HTML-формы
- Google reCAPTCHA v2

## Установка

1. Склонировать репозиторий:
git clone https://github.com/anton-tt/php_auth_app
2. Настроить PostgreSQL:
- создать базу данных `php_auth_db`
- создать таблицу `users` (см. директорию `migrations`)
- настроить подключение в `config/db.php`
3. Зарегистрировать сайт в Google reCAPTCHA и получить:
- Site key
- Secret key  
Подставить ключи в `login.php`.
4. Запустить встроенный сервер PHP:
```bash
php -S 127.0.0.1:8000 -t public
```

## Использование
Перейти в браузере на главную страницу `index.php` .
Неавторизованным пользователям доступны:
- регистрация
- вход в аккаунт.

После авторизации пользователь может:
- просмотреть профиль
- обновить личные данные
- изменить пароль
- выйти из аккаунта.
